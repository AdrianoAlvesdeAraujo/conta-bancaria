#   Conta Bancária :bank:

programa em JAVA de um Banco. Temos duas classes, sendo  a principal, a classe **Main** e a classe **ContaBanco**. 

### Classe ContaBanco

#### Atributos:

- numConta - tipo  inteiro;
- tipo - tipo String
- saldo - tipo double;
- dono - tipo String;
- status - tipo boolean;

#### Métodos:

- abrirConta();
- fecharConta();
- pagarMensal();
- sacar();
- depositar();
- status();
- getter e setter de todos os atributos;
